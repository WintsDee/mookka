## Getting Started

### Prerequisites
- Node.js (recommended: latest LTS version)
- npm or yarn

### Installation Steps

```bash
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to project directory
cd <PROJECT_NAME>

# Install dependencies
npm install

# Start development server
npm run dev
```
